# Batch Image Cropping and Resizing for creating Looking Glass Quilts
A .Net Windows Desktop Application for Cropping and Scaling a batch of images. Exports Looking Glass Quilts

## Description
Upon finding myself with a sequence of images from a Camera Rail and seemingly NO free offline batch image editors I hacked this app together using .Net 4.8.
It allows you to Crop, Scale and export a sequence of images and to export them as Looking Glass Quilts.
